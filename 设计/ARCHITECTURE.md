## 数据流向图
用户提问 "失眠怎么办"
│
▼
┌───────────────────────────────────────────────────────────────┐
│ 查询改写: "失眠的改善方法和治疗方案" │
│ 查询扩写: ["失眠怎么办", "失眠治疗方法", "改善睡眠的建议"] │
└───────────────────────────────────────────────────────────────┘
│
▼
┌───────────────────────────────────────────────────────────────┐
│ 并行检索（同时进行） │
│ Milvus: 返回10条语义相关文档 │
│ BM25: 返回10条关键词匹配文档 │
│ MySQL: 返回5条历史相似对话 │
│ Redis: 返回0条缓存（首次查询） │
│ 互联网: 返回3条实时搜索结果 │
└───────────────────────────────────────────────────────────────┘
│
▼
┌───────────────────────────────────────────────────────────────┐
│ RRF融合: 28条结果 → 去重 → 排序 → 取TOP3 │
└───────────────────────────────────────────────────────────────┘
│
▼
┌───────────────────────────────────────────────────────────────┐
│ DeepSeek: 基于检索知识 + 心理医生角色 + 默认人格 → 生成回复 │
└───────────────────────────────────────────────────────────────┘
│
▼
┌───────────────────────────────────────────────────────────────┐
│ 流式输出: 一个字一个字显示 → 用户看到回复 │
└───────────────────────────────────────────────────────────────┘

## 技术栈

| 层次 | 技术 | 作用 |
|------|------|------|
| Web框架 | FastAPI + Uvicorn | API服务 |
| 向量检索 | Milvus + BGE-M3 | 语义搜索 |
| 关键词检索 | BM25 + jieba | 字面匹配 |
| 缓存 | Redis | 热数据缓存 |
| 数据库 | MySQL | 用户/收藏/历史 |
| 重排序 | BGE-Reranker-v2-m3 | 结果精排 |
| 大模型 | DeepSeek API | 对话生成 |
| 前端 | HTML/CSS/JS | 用户界面 |
| 部署 | Docker | Milvus容器 |
